# ExploreEase – Travel Guide Website
Welcome to ExploreEase, a simple and beautifully designed Travel Guide Website created using HTML, CSS, and JavaScript.

2) Plan Trip Page – Create a travel plan
3) About Page – Learn about the website
4) Contact Page – Send us a message
5) Login & Signup Page – Authentication UI
## Technologies Used
1) HTML5
2) CSS3
3) JavaScript
4) Bootstrap
